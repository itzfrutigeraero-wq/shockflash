# Games Directory

Add your classic Flash game SWF files here.

The website will automatically detect and display any .swf files placed in this folder.

## How to Add Games

1. Download SWF files from:
   - [Internet Archive Flash Games](https://archive.org/details/softwarelibrary_flash_games)
   - [Flashpoint Archive](https://flashpointarchive.org/)
   - [GitHub Flash Games Collections](https://github.com/AmmarSAA/Flash-Games-Directory)

2. Place your `.swf` files directly in this `/games/` folder

3. Update `js/script.js` with the game details:
   ```javascript
   {
       id: 1,
       title: "Game Name",
       developer: "Developer",
       year: 2010,
       category: "action", // action, puzzle, adventure, sports, strategy, casual
       description: "Game description",
       emoji: "🎮",
       swf: "games/filename.swf"
   }
   ```

4. Refresh your browser - your game appears instantly!
