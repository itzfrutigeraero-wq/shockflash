About SHOCKFLASH

SHOCKFLASH is a museum of classic Flash games from the golden era of web gaming. Using the Ruffle emulator, we can play these archived SWF files in modern browsers without the need for the deprecated Flash Player plugin.

What is Shockwave Flash (SWF)? SWF stands for Small Web Format and was the file format used by Adobe Flash to deliver interactive content, animations, and games on the web from the 1990s through 2020s.

What is Ruffle? Ruffle is an open-source Flash Player emulator written in Rust and WebAssembly, allowing Flash content to run in modern browsers safely and without proprietary software.
